# 🧮 Python Basic Calculator

A simple command-line calculator built in Python that performs basic arithmetic operations: addition, subtraction, multiplication, and division.

## 🚀 Features
- Supports `+`, `-`, `*`, `/` operations  
- Handles invalid (non-numeric) input  
- Prevents division by zero  
- Continuous operation until user exits  

## 🛠️ Technologies Used
- Python 3  

## ▶️ How to Run
1. Ensure Python is installed  
2. Save the file as `calculator.py`  
3. Run the script:
```bash
python calculator.py
